Ruby Variable Scopes

  • local_variable
    Visible in the scope its defined.
  • $global_variable
    visible everywhere
  • CONSTANT_VARIABLE
    also has global scope
    It's basically the same thing as a global variable, however, once you define it, you cannot redefine
  • @instance_vairable
    Visible only to a single instance of a class
  • @@class_variable
    Visible to ALL instances of the SAME class

Ruby Error Messages

  • unexpected keyword_end, expecting end-of-input
    you have an end in there that doesn't belong. Either the beginning of the block is wrong or it's an extra end
  • unexpected tSTRING_BEG, expecting end of input
    it was expecting the beginning of a string, so a single or double quote
  • unexpected token
    extra character in there that shouldn't be there
  • undefined method 'length' for nil:NilClass (NoMethodError)
    trying to call a method, in this case length, on a variable that evaluates to nil.
  • private method 'append_feat' called for array
    you can just call the method on its own
  • no implicit conversion of Fixnum into string (TypeError)
    you're trying to add a string with an integer, but there's no way to do it
