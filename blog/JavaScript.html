JS vs Ruby

Ruby class v. JavaScript constructor functions

Constructor functions are different from classes in ruby because classes have a particular behavior that is set by multiple methods. Classes have lots of different properies that all get adjusted and manipulated by unique methods to that class.

A Constructor function is the template for one function, and when it's called, it'll set up a new version of that function.
